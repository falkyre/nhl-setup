Run diagnostic and helper tools.

Upload Logs for GitHub Issue

This will run the issueUpload.sh script to collect your configuration and logs, and upload them to a private pastebin. This is useful for troubleshooting when opening an issue on GitHub.

Download Configuration

Download your config.json file.

If you are running on a the Scoreboard image V2025.12.1 or later, this will download a zip file with config.json, the scoreboard.conf service file, the splash.sh and testMatrix.sh scripts.

The configs.zip file can be placed in the /boot/scoreboard directory on newer images (V2025.12.1 and later) to auto-deploy the configuration on boot.
